# PyVeritas
Intelligent Python software testing.

# Modalities
The following modes of operation exist.

## Declarative Tests
Simple suite.test() statements for traditional unit tests.

## AI-Powered Fuzzing
Customizable fuzzing using VeritasFuzzer.

## Behavior Modeling
Use VeritasBehavioralModel to validate against expected behaviors.

## Environment-Aware Testing
Easily test compatibility across Python versions or platforms.

## Actionable Feedback
Gather insights on test coverage, bottlenecks, and failure clusters.
